About Me

Work

Previously, I worked as a fullstack web developer. More specifically, I spent (approximately) 75% of my time writing Python microservices for deployment in a Linux environment in AWS, and the other 25% creating single page web applications in Angular. I also performed various devops-related tasks on an ad-hoc basis.

Home

Information Security is my passion. As such, I spend much of my time outside of work engaging in tasks to further my InfoSec knowledge and build my skills. These include:

  • Linux and Windows system programming in C
  • Pen-testing machines and completing challenges in the Hack the Box labs
  • Creating and contributing to various open source projects
  • Building, modifying, and pen-testing my home lab network

Most recently, I completed the Offensive Security Penetration Testing with Kali Linux (PWK) course and obtained my Offensive Security Certificated Professional (OSCP) certification.
